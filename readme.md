## Puzzle Game

* Fork the repository

* Clone the repository

```bash
git clone https://github.com/<username>/puzzle-game.git
```

* Go to the directory `puzzle-game`

```bash
cd puzzle-game/
```

* Compile the file `main.c` and create the executable `main`

```bash
gcc main.c -o main
```

* Run the game `main`

```bash
./main
```
